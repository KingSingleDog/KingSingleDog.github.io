👾 git常用到的命令

无脑操作（新手专属）

# 克隆仓库到本地
git clone [仓库地址]

# 打开本地仓库
cd [仓库名]

# 修改文件
......

# 添加全部修改过的内容到暂存区
git add --all

# 提交修改
git commit -m "本次提交的说明"

# 推送到远程仓库
git push -u origin master

下面的是更多的一些命令，事实上，你只要在命令行输入git就可以查看全部git命令了

新建仓库

# 在本地创建一个名为[项目名称]的仓库
git init [项目名称]

# 克隆仓库到本地
git clone [仓库链接]

配置

# 显示配置列表
git config --list

# 修改配置中的用户名
git config [--global] user.name "[用户名]"

# 修改配置中的邮箱地址
git config [--global] user.email "[邮箱地址]"

增加/删除文件

# 添加当前工作区的全部文件到暂存区
git add --all

# 添加指定文件到暂存区
git add [文件1] [文件2] [文件3]

# 添加文件夹到暂存区，包括子目录
git add [文件夹]

# 添加当前目录到暂存区
git add .

# 添加每个变化前都会要求确认
git add -p

# 删除工作区的指定文件并把这次删除放入暂存区
git rm [文件1] [文件2] [文件3]

# 停止追踪指定文件，但该文件会保留在工作区
git rm --cached [文件]

# 移动文件，可用来重命名文件
git mv [原文件] [移动后的文件]
